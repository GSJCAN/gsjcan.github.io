趁着睡不着的时间写点东西

一半忙碌

暑假理论上是从6月24日中午开始的，这天早上考完大三最后一科笔试《量子力学》。愚昧地认为所有笔试考完是假期的开始。

至于一些课程布置的其它作业，只要能在deadline之前交完自然就没问题，邮件提交当然没有地点的限制，所以觉得自己获得了自由，支配时间的自由。虽然我还是得在学校待上几天。

之后还有一个《物理教学论》的课程论文要交，还有一个编写教材的任务，索性花的时间不太多。然后还有一个量子力学的大作业，当然也不难，很快就完成了。一开始觉得困难没法下手的事情，到最后发现好像并没有这么困难。

在29号之前搞定所有的需要在学校做完得事情之后，开始了我在北京的Summer Camp。十分耗费人力和财力。尽管结果并不如人意，但至少活了20年，总算去过了一趟北京以及TOP 2的学校。

PKU：说实话，我真是很喜欢Louis Tao，也对Computational Neuroscience非常感兴趣，但是我感觉Tao并不怎么对我感兴趣，还把我推给了另外一个PI，之后的邮件也回得十分敷衍，奈何自己在个人展示的时候搞得太水，好吧自己也的确比较水；

THU：在THU的日子里我主要是被学校吸引，可能自己在来THU之前没有深入了解，所以并没有抱着很明确的目标而来，也就准备地一般。我十分喜欢THU的校园，和PKU那种比较沉闷的气氛完全不同，THU的校园让我觉得非常自在，可能是绿化比PKU做得好。和几个同行的同学都说非常希望可以留下来啊。（当然，清华两个字也是非常好听的，比另外一所TOP 2的名字好听多了。比USTC的名字也好听。不过，THU本身没USTC好看。）非常羡慕LC同学，祝福他；

NJU：本身不是特别喜欢这个学校，主要是因为名气去的。NJU主要做凝聚态，但我早已决定不搞凝聚态，所以也是没有好好准备。最后面试的是声学专业，问了几个问题基本回答了上来，不过翻译有点紧张，没发挥好。Anyway, let it go；

USTC：由于我在去USTC之前就已经过了网面以及线下面试，所以其实Summer Camp主要是冲着找老师和课题组确定去向而去的，所以这几天过的也就比较轻松。由于网面拿了A，本不用参加笔试，但是被忽悠还是去考试了，事实证明此举十分不明智，因为没怎么复习，给地大丢脸了好吧。苑老师在冷餐会之前就已经做了一场报告，见到男神还是十分激动的。冷餐会终于可以和苑老师交流，由于两个面试都通过了，当然我也就很顺利地进入了大组，这样我之后几天就可以说是非常轻松了。（此处感谢苑老师。）可能由于人品大爆发，又在之后的毕营晚会上被抽中一个玩偶，大概是5/1000的概率吧（虽然没有拿到大奖签名T恤，但是既然签名没有男神潘老师的，要来有何用，不稀罕好吧哈哈哈）；

BNU、ZJU、IHEP、WIPM、WNLO、Brain-BNU：很遗憾这几个地方虽然通过了但是最终没有去参加夏令营主要是时间冲突以及囊中羞涩，无法去参加这些地方的活动，所以只能在众多的优秀学校中做出选择。写拒绝邮件的时候其实我是很痛心的。但夏令营本就是一个学校与学生互相选择的过程，学校选择我，我选择学校，学校再面试我，我再选择offer，如此多次循环。有缘再见吧。

一半清闲

在7月222日USTC夏令营结束之后，回校学校待了几天。25号回家，发觉今天也是25号，一个月过去了。

下火车出站立马就见到了殷姑娘，一件粉红色T恤，一条牛仔半身裙，一双白色匡威开口笑布鞋，美过十里春风。由于想和殷姑娘多待一会，故意和父母把自己下火车的时间说后了半个多小时。然后在旁边的KFC吃了点东西，薯条自然是必备的，半年不见自然是非常想念。

回到韶关的时间基本上是在家待一天，出去和殷姑娘呆一天。陪殷姑娘去自习（我是天天迟到的那个…），督促她好好复习考研，以及一起商量考研的学校，复习计划。如今考研临近，复习时间越来越短，希望她能够好好加油，考研成功。我认为考研是一件十分辛苦的事情，曾经兄弟和我说，他想好好享受大学的前两年多，然后最后好好努力复习考研，从而不用为了要保研而刷GPA，刷各自经历。当然，我认为考研也要看经历的吧。我当时觉得这种选择也是有意义的（现在也觉得），但是现在觉得考研复习实在是煎熬，我可能熬不下去。

考研路上都是勇士，兄弟是勇士，殷姑娘也是勇士。祝福殷姑娘。

在家自然是很无聊的，由于各种原因。

期间回老家了一趟，我是非常喜欢回老家的，度过人生前15年的地方。非常怀念我大伯，大娘。回家当晚便把offer以及研究生的大概去向以及7月的经历跟大伯以及其它亲人讲，大伯和家人显得非常高兴，不过也表达了我没能去成THU的遗憾。我也稍微为自己辩解了一下，用了什么名额少，牛人多之类的借口，当然我内心也为自己的水感到非常愧疚，要是我能去THU大伯也就能更高兴了。尽管潘组并不比THU的各个课题组差，我自己也非常非常喜欢潘组，但是不得不承认中国目前的老一辈人还是对THU的这几个字无比地向往，或者说在非内行人看来，THU的确比什么USTC要厉害多了。

然后8月17日回到武汉，宿舍台式电脑坏了，游戏全没得玩，还我守望先锋！

在各个宿舍拉了几个人，有空就玩玩三国杀。我对于很多游戏的看法依然是觉得要和认识的人一起玩才有意思，一个人玩没啥意思，比如线上三国杀，王者荣耀，以及各种单机可多人游戏等等（守望先锋除外）。
